Whether you use Google Gemini (cloud) or Ollama (local), Prometheus bridges the gap between natural language and the command line — with built-in safety checks, a rich history system, and an extensible plugin architecture.

## What Prometheus does

Type what you want — Prometheus figures out the command, shows it to you, and runs it. Dangerous operations get a safety warning before anything happens.

```bash
prom "find all python files larger than 1MB"
prom "show disk usage sorted by size"
prom "create a backup of my documents folder"
```

<Steps>
  <Step title="Install Prometheus">
    Clone the repository and run the system installer — it sets up a shell alias (`prom` by default) and walks you through choosing an AI backend.

    ```bash
    git clone https://github.com/roywalk3r/prometheus.git
    cd prometheus
    sudo bash system-install.sh
    ```
  </Step>
  <Step title="Configure your AI model">
    Use Google Gemini (recommended — free tier, fast) or a local Ollama model for full privacy. Set your API key and you're ready.

    ```bash
    export GEMINI_API_KEY="your-key-here"
    ```
  </Step>
  <Step title="Start using natural language">
    Launch interactive mode with `prom` or pass a query directly. Prometheus generates the shell command, displays it, and asks for confirmation before running.

    ```bash
    prom "list all git branches sorted by last commit"
    ```
  </Step>
</Steps>
